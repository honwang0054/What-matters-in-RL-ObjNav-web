# What Matters in RL-Based Methods for Object-Goal Navigation? — Project Page

🌐 **Live page:** https://honwang0054.github.io/What-matters-in-RL-ObjNav-web/
📄 **Paper (arXiv):** https://arxiv.org/abs/2510.01830
💻 **Code:** https://github.com/honwang0054/What-matters-in-RL-ObjNav-release

[Nerfies](https://github.com/nerfies/nerfies.github.io) project page template
